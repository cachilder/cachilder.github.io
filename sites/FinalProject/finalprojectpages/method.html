Technologies Used

The visualizations created on this site use a program called Voyant. This program was chosen for ease of use, wide availability, and price considerations. Voyant is currently available for free on any web based application. There are a variety of visualizations available, however the primary ones used here include TermsBerry and Labyrinth. The data from these projects can also be exported and shared with others through text files directly from Voyant.
